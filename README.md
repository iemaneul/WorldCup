# Fifa World Cup

Aplicativo browser para acompanhar a Copa do Mundo com navegação fluida entre páginas, no estilo de app mobile.

## Sobre o projeto

Este projeto foi criado como uma experiência web leve para acompanhar os principais pontos de uma Copa do Mundo em uma interface simples e direta.

O app inclui páginas dedicadas para:

- início
- jogos
- grupos
- seleções
- países-sede

## Experiência

- navegação inferior flutuante
- troca entre páginas sem recarregamento completo
- estrutura componentizada para facilitar manutenção
- organização de assets em pastas separadas de CSS e JavaScript

## Estrutura

```text
.
├── index.html
├── jogos.html

├── paises-sede.html
└── static
    ├── css

    └── js

        └── nav-component.js
```

## Objetivo

Servir como base para um aplicativo web visual e responsivo focado em acompanhar os jogos da Copa do Mundo diretamente no navegador.
